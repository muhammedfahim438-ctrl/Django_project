# Django Task Manager Project

A complete Django Task Management web application with user authentication, profile image upload, task management dashboard, and REST API support.

---

# Features

## User Authentication
- User Registration
- User Login
- User Logout
- Secure Password Storage

## User Profile
- Upload Profile Photo
- Update User Details
- Manage Account

## Task Management
- Add New Task
- Edit Task
- Delete Task
- Mark Task Complete / Pending
- Task Dashboard

## User Management
- View All Users
- Delete Users

## REST API
- JWT Authentication
- User CRUD API
- Task CRUD API
- Image Upload API

## Admin Panel
- Django Admin Access
